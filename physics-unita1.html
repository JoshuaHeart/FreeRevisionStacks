Kinematics

Comprehensive notes on motion analysis based on the IB Physics HL Syllabus.

1. Basic Definitions

Distance (\(d\)): Scalar. Total path length travelled. (Always \(\ge 0\)).

Displacement (\(s\)): Vector. Straight-line change in position from start to end.

Speed (\(v\)): Scalar. \(\text{dist} / \text{time}\).

Velocity (\(v\)): Vector. \(\Delta s / \Delta t\).

Acceleration (\(a\)): Vector. Rate of change of velocity (\(\Delta v / \Delta t\)).

The 400m Runner Example

Distance = 400m. Displacement = 0m (returned to start).

3. SUVAT (Constant Accel.)

\(v = u + at\)
From \(a = (v-u)/t\).
\(s = \frac{(u+v)t}{2}\)
Average velocity \(\times\) time.
\(s = ut + \frac{1}{2}at^2\)
Combines the two above.
\(v^2 = u^2 + 2as\)
Eliminates 't' from the set.

4. Vector Decomposition

How it works:

In physics, we use SOH-CAH-TOA to split any diagonal vector into its horizontal and vertical components.

  • Adjacent (\(x\)): Uses \(\cos \theta\) because it's "next to" the angle.
  • Opposite (\(y\)): Uses \(\sin \theta\) because it's "across" from the angle.

5. Terminal Velocity (Skydiver)

Forces on a Skydiver

The Process:

  1. Initial Fall: Velocity is low, so Drag is low. Acceleration \(\approx g\).
  2. Acceleration: As velocity increases, Drag (\(D\)) increases.
  3. Net Force Decreases: \(F_{\text{net}} = W - D\). As \(D\) grows, \(F_{\text{net}}\) shrinks.
  4. Terminal Velocity: Eventually, \(D = W\). \(F_{\text{net}} = 0\), so \(a = 0\). The skydiver falls at a constant speed.
